## 📌 About the Project

Selecting the right crop is one of the most important decisions in agriculture. This project uses **Machine Learning** to analyze soil nutrients and environmental conditions, helping users identify crops suitable for a given set of conditions.

The project applies **K-Means Clustering**, an unsupervised learning algorithm, to group similar agricultural conditions and recommend crops based on the predicted cluster.

The application is built with **Streamlit**, providing an interactive and user-friendly interface.

- Rainfall
